ICT ontwerpt EMS

Intro

Met 6 Industriële ICT zochten we een uitdagend en maatschappelijk relevant project. Een project waarin we onze kennis en vaardigheden zouden nodig hebben om een maatschappelijk relevant probleem aan te pakken. We vonden dit in het ontwerpen van een eigen energiemanagementsysteem.

Doelstelling

We brengen de energieconsumptie in kaart en bekijken op welke manieren we de consumptiepatronen kunnen optimaliseren.

Werkwijze

Op de Raspberry Pi hebben we een Linux server opgezet. We verzamelen er data met behulp van Pythonscripts en Bash. Om de data toegankelijk te maken via het schoolnetwerk hebben we een RESTful API opgezet met Python. De frontend is opgebouwd met HTML en CSS. Voor het opvragen en visualiseren van de data maken we gebruik van JavaScript.

Toekomstvisie

Ons energiemanagementsysteem is zopas in gebruik genomen in dit gebouw. Volgend schooljaar willen we met 6ICT de opbouw van het huidig systeem bestuderen en bijsturen. Vervolgens willen we de installatie en mogelijkheden uitbreiden. We denken aan:

  • Energieproductie met zonnepanelen
  • Slim sturen van een autolader
  • Opzetten van een waarschuwingssysteem
  • …
